# 🍽️ Hôm Nay Ăn Gì?

> 🤔 Ứng dụng gợi ý món ăn ngẫu nhiên – giải quyết câu hỏi muôn thuở: *“Hôm nay ăn gì?”*

🌐 **Live Demo:** https://panbap.github.io/Homnayangi/

---

## 🚀 Giới thiệu

Đây là một project **Hôm Nay Ăn Gì** được xây dựng bằng HTML, CSS và JavaScript.

Ứng dụng giúp bạn chọn món ăn một cách ngẫu nhiên khi bạn không biết nên ăn gì.

💡 Chỉ cần một click → có ngay gợi ý!

---

## 🎯 Cách hoạt động

- Nhấn nút để chọn món ăn  
- Hệ thống sẽ random món ăn  
- Hiển thị kết quả ngay lập tức  
- Có thể chọn lại nhiều lần  

---

## ✨ Tính năng

- 🍜 Gợi ý món ăn ngẫu nhiên  
- ⚡ Phản hồi nhanh, không cần tải lại

- 🎨 Giao diện đơn giản, dễ dùng
